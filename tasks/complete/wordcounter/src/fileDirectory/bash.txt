        Настройка Командной Строки
Командная строка-это мощный инструмент, используемый разработчиками для поиска, создания и управления файлами и папками. В этом коротком руководстве вы узнаете, как настроить приложение командной строки на вашем компьютере.

Интерфейсы командной строки (CLIs) бывают разных форм. CLI, который мы будем использовать, называется Bash.

Что такое Баш?
Bash, или Bourne-Again SHell, - Это клип, созданный более двадцати семи лет назад Брайаном Фоксом в качестве замены свободного программного обеспечения для Bourne Shell. Оболочка-это особый вид CLI. Bash - это "открытый исходный код", который означает, что любой человек может прочитать код и предложить изменения. С самого начала он был поддержан большим сообществом инженеров, которые работали над тем, чтобы сделать его невероятным инструментом. Bash-это оболочка по умолчанию для Linux и Mac. По этим причинам Bash является наиболее используемой и широко распространенной оболочкой. Если вы хотите узнать больше о Bash, эта статья в Википедии-хорошее место для начала.

Из- за силы сообщества с открытым исходным кодом и инструментов, которые они предоставляют, освоение Bash- это лучшая инвестиция, чем освоение командной строки .

Попробуйте это сделать!
Теперь, когда на вашем рабочем столе открыт интерфейс командной строки, вы готовы его использовать. Попробуйте выполнить некоторые команды на своем персональном компьютере. Вот несколько хороших команд для практики:
ls для отображения содержимого текущего каталога. Это может выглядеть примерно так:
$ ls
Applications                Pictures
Codecademy                  Public
Desktop                     Downloads
Documents                   Library

 mkdir test для создания нового каталога с именем test. Теперь, когда вы набираете ls, вы должны увидеть папку с именем test:
$ ls
Applications                Pictures
Codecademy                  Public
Desktop                     Downloads
Documents                   Library
test

cd test для перехода в новый каталог. 
echo "Hello Command Line" >> hello_cli.txt
чтобы создать новый файл с именем hello_cli.txt и добавьте в этот файл командную строку Hello. При вводе ls вы должны увидеть следующее:
$ ls
hello_cli.txt
cat hello_cli.txt для печати содержимого hello_cli.txt- файл на терминал. Вы должны увидеть что- то вроде:
$ cat hello_cli.txt
Hello Command Line

Хорошая работа! Вы готовы исследовать мир интерфейса командной строки на своем собственном компьютере.
